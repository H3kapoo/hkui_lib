\doxysubsection{Detailed Description}
Singleton class responsible for loading fonts. 

Class that loads fonts into memory using vendor/stb\+\_\+image and caches data for later use so that two calls to load the same font will result in cache being accessed instead.

\begin{DoxyNote}{Note}
Currently this is not hot reloadable like shaders/textures. 
\end{DoxyNote}
