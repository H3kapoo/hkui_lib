\doxysubsection{Detailed Description}
Basic rectangular mesh class encapsulation. 

Class representing a rectangular mesh who\textquotesingle{}s shader can be set and transforms changed. Contains subclass \mbox{\hyperlink{structmeshHelpers_1_1RectMesh_1_1BoxModel}{Box\+Model}} representing translation/rotation/scale of the rectangular mesh.
