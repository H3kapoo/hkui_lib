\doxysubsection{Detailed Description}
Singleton providing shader load/reload and uniform access capabilities. 

Singleton class providing easy means to load/reload shaders and access to their uniforms. Class also provides caching capabilities for shaders.

{\footnotesize\ttfamily shader\+Id\+Ptr shader\+Helpers\+::\+Shader\+Helper\+::load\+From\+Path (\begin{DoxyParamCaption}\item[{const std\+::string \&}]{vert\+Path,  }\item[{const std\+::string \&}]{frag\+Path }\end{DoxyParamCaption})}



Load and retrieve a shader {\bfseries{Id}} pointer handle from the specified vertex and fragment shader paths. 

This function checks if a shader with the specified vertex and fragment shader already exists in the shader cache. If it does, it returns a pointer to its {\bfseries{Id}}. Otherwise, it compiles and links the shaders, stores the {\bfseries{Id}} in the cache, and returns a pointer to the new {\bfseries{Id}}.

\begin{DoxyNote}{Note}
Returned dereferenced pointer value will be -\/1 in case any errors occured. It will never be {\bfseries{nullptr}}. 

Reason for pointer return instead of plain integer is to have easier shader reloading capabilities.
\end{DoxyNote}
