Load and retrieve a texture {\bfseries{\mbox{\hyperlink{structtextureHelpers_1_1TextureData}{Texture\+Data}}}} pointer handle from the specified texture path. 

This function checks if a texture with the specified texture path already exists in the textures cache. If it does, it returns a pointer to its {\bfseries{\mbox{\hyperlink{structtextureHelpers_1_1TextureData}{Texture\+Data}}}}. Otherwise, it loads the texture, stores the loaded {\bfseries{\mbox{\hyperlink{structtextureHelpers_1_1TextureData}{Texture\+Data}}}} in the cache, and returns a pointer to the newly loaded {\bfseries{\mbox{\hyperlink{structtextureHelpers_1_1TextureData}{Texture\+Data}}}}.

\begin{DoxyNote}{Note}
Returned dereferenced pointer value {\bfseries{g\+Id}} will be 0 in case any errors occured. It will never be {\bfseries{nullptr}}. 

Reason for pointer return instead of plain structure is to have easier texture reloading capabilities.
\end{DoxyNote}
