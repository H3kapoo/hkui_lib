\doxysubsection{Detailed Description}
Class used to speed-\/up searches in the default \mbox{\hyperlink{classtreeHelpers_1_1TreeStruct}{Tree\+Struct}}. 

Speeds up tree searching by flattening and sorting the tree into an array according to the supplied function inside {\bfseries{sort\+Buffer}}. Internal flat vector needs to be manually asked to sort itself, it doesn\textquotesingle{}t do that autonomously to allow user to also sort tree in batches. It is recommended to ask for resorting after each single/bached node addition/removal in order to avoid UB.

\begin{DoxyNote}{Note}
This should really only be instantiated on the {\bfseries{Root}} node of the tree. 
\end{DoxyNote}
