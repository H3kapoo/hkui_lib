vaoId meshHelpers::MeshBuilder::genQuadMesh ( )

Generate vao Id for a simple quad mesh.

Generate vertex data for a quad that's using an index buffer. It has two attributes: position and texture coordinates. If quad vao has been generated before, simply return it's cached value.

Returns
vaoID of generated quad mesh to be bound.
