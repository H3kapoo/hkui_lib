Detailed Description

Basic rectangular mesh class encapsulation.

Class representing a rectangular mesh who's shader can be set and transforms changed. Contains subclass BoxModel representing translation/rotation/scale of the rectangular mesh.

shaderHelpers::shaderId meshHelpers::RectMesh::getShaderId ( ) const
inline

Get the object's shader Id.

Get object's shader Id and, if not set, return 0 denoting no shader has been computed for this object.

Note
Note that this is a pure value returned, not a reference type. If the shader of the object is reloaded, consumer needs to call getShaderId again to get the after-reload new Id.
Returns
Shader Id of the current object.

◆ getShaderIdPtr()

shaderHelpers::shaderIdPtr meshHelpers::RectMesh::getShaderIdPtr ( ) const
inline

Get the object's shader Id pointer.

Get object's shader Id pointer. It may be nullptr (very unlikely) if not initialized or -1 if shader loading failed.

Note
If shader of object tends to frequently reload, it's better to use this function to get the Id instead of getShaderId.
Returns
Shader Id pointer of the current object.

◆ getTransform()

glm::mat4 meshHelpers::RectMesh::getTransform ( )

Get model matrix of object.

Compute and return the model matrix of this object. Scale and Translation is relative to top left corner and Rotation is relative to the center of the rectangle.

Note
Off by one pixel render defects might occur due to rotation being applied from the center instead or from the top left corner.
Returns
Mat4x4 representing OpenGL compliant model matrix.
