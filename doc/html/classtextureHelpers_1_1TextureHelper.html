TextureDataPtr textureHelpers::TextureHelper::loadTexture ( const std::string &  filePath)

Load and retrieve a texture TextureData pointer handle from the specified texture path.

This function checks if a texture with the specified texture path already exists in the textures cache. If it does, it returns a pointer to its TextureData. Otherwise, it loads the texture, stores the loaded TextureData in the cache, and returns a pointer to the newly loaded TextureData.

Note
Returned dereferenced pointer value gId will be 0 in case any errors occured. It will never be nullptr.
Reason for pointer return instead of plain structure is to have easier texture reloading capabilities.
Parameters
filePathThe file path to the texture.
Returns
A pointer to the texture data structure.

void textureHelpers::TextureHelper::reloadFromPath ( const std::string &  filePath)

Reload a texture from the specified texture path.

This function creates a new TextureData from the provided texture path and overrides the cached value at filePath location with the newly created TextureData if successfull, otherwise it keeps previous data.
