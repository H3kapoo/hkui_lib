Detailed Description

Abstract base class representing rectangle node.

Class used to be inherited from to create new types of nodes sharing basic event/structure functionality.

void treeHelpers::RectNodeABC::setEventTransparent ( const bool  value)

Set node to be transparent to any events that might occur on it.

Set node to be transparent to any events that might occur on it like hover, click, etc. The next valid node after this one will receive the event instead

Note
Currently this makes the node transparent to ALL events. In the future it will be better to make it transparent only to certain events.
Parameters
valueBoolean deciding if node should be transparent or not.
